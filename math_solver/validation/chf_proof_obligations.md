# CHF Proof Obligations

## PO-001 — Radial Coverage

If `Ω` is compact and star-shaped with respect to center `S`, and its boundary is partitioned into patches `B_i`, then inward radial cells from `S` to each `B_i` cover `Ω`.

## PO-002 — Category Change

A transition point cannot become an actual next state unless a crossing event occurs.

```text
potential -> crossing -> actual
```

## PO-003 — Conservative Uncertainty

If multiple plausible centers remain, robust `ALLOW` requires the proposed transition to pass under all represented plausible centers. Failure under any unresolved plausible center blocks robust permission.

## PO-004 — Local-Global Coupling

Local `ALLOW` does not imply global `ALLOW`. Affected-cloud deformation and affected-entity recoverability must remain within threshold.

## PO-005 — Threshold Record

Every actualized crossing produces:

```text
Σ = historical shell
ρ = propagated record
```

Record existence and record legibility are separate.

## PO-006 — Observer Projection

The same radial tessellation may appear smooth or cell-resolved depending on observer distance, resolution, noise, lag, and probe capacity.

## PO-007 — Legibility Decay

A propagated record may persist while its legibility falls below reconstruction threshold. A historical event can remain causally real while becoming epistemically degraded.
